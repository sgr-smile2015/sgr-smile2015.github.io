ubuntu配置jenkins环境

发表于 2017-06-30

ubuntu

前言

  • 在jenkins2.60.1版本后java版本最少需要1.8了,所以需要更新java版本;
  • jenkins2.0以后引入了pipeline流水线功能,这个很赞;
  • 2.60貌似也是长期支持的版本;

安装java1.8

  • 下载好安装包,jdk-8u112-linux-x64.tar.gz,去oracle官网下载,有点慢,没有办法;
  • 解压到/use/lib/jvm目录下,重命名java-8-oracle;
  • 修改目录的权限sudo chown -R root:root /usr/lib/jvm/java-8-oracle;
  • 最后使用update-alternatives命令设置系统默认的java环境;

安装完成验证:java -version

java version “1.8.0_131”
Java(TM) SE Runtime Environment (build 1.8.0_131-b11)
Java HotSpot(TM) 64-Bit Server VM (build 25.131-b11, mixed mode)

配置jenkins的主目录,在ubuntu14.04的配置文件在/etc/default/jenkins这个文件

# ubuntu,jenkins-ci
